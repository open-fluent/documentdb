# Fluent.DocumentDb

Fluent.DocumentDb is the core .NET package for building applications on top of [DocumentDB](https://github.com/documentdb/documentdb), an open-source document database built on PostgreSQL.

DocumentDB brings document database workloads to the PostgreSQL ecosystem. It is designed as an open, portable, PostgreSQL-backed alternative for teams that need MongoDB-style development without committing to a single vendor database platform.

## Why this package

- Use a document database model backed by PostgreSQL
- Build toward MongoDB-compatible workloads and tooling
- Keep the operational benefits of PostgreSQL and its mature JSON/JSONB ecosystem
- Avoid tight coupling to proprietary document database services
- Prepare .NET applications for DocumentDB-based storage with a small, focused package

## Install

```powershell
dotnet add package Fluent.DocumentDb
```

## Status

This package is in early development. The API is being shaped around compatibility, performance, and a simple developer experience for modern .NET applications.

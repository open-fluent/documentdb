# 🚀 DocumentDB .NET Client

DocumentDB is a .NET client library for Microsoft DocumentDB — a PostgreSQL-compatible database with advanced JSON document storage and querying capabilities. The client provides high-performance access to DocumentDB features directly from .NET applications, allowing developers to work with relational and document data seamlessly.

## 📦 What is this repository?

This repository contains the official .NET client SDK for RedFoxDB. It provides:

- Full support for DocumentDB document extensions over PostgreSQL.
- Native integration with PostgreSQL features using Npgsql.
- Strongly-typed document operations.
- Efficient serialization via `System.Text.Json`.
- Fully async API surface for scalable applications.

## 🚀 Getting Started

The package is available on NuGet:  
<https://www.nuget.org/packages/Fluent.DocumentDb>

Install via .NET CLI:

```powershell
dotnet add package Fluent.DocumentDb
```

Or via Package Manager Console:

```powershell
NuGet\Install-Package Fluent.DocumentDb
```
